In-air - simple fly tickets system (boilerplate).
=====================

## What included

 - Angular 1.5.6
 - Gulp build toolchain (webpack, ES6 - babel)

### Front-end
Just type:
* `gulp` or `gulp serve` for dev mode
* `gulp --prod` or `gulp --production` for build an optimized app and transfer to `web` directory
